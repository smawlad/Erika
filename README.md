# Erika

Erika is a simple command-line social network.

## Dependencies

### python 3.7.7 or above

Instructions for how to install Python can be found [here.](https://www.python.org/downloads/)

### flask

```bash
pip install Flask
```

### flask-mysqldb

```bash
pip install Flask-MySQLdb
```

### requests

```bash
pip install requests
```

## To run

```bash
python api/api.py
```
```bash
python app.py
```
